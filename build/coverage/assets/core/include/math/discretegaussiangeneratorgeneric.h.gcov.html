      37             : /*This is the header file for the Generic Sampler used for various Discrete
      38             :  * Gaussian Sampling applications. This class implements the generic sampler by
      39             :  * UCSD discussed in the https://eprint.iacr.org/2017/259.pdf and it is heavily
      40             :  * based on Michael Walter's original code. Along the sides of the
      41             :  * implementation there are also two different "base samplers", which are used
      42             :  * for the generic sampler or can be used on their own depending on the
      43             :  * requirements of needed application.
      44             :  *
      45             :  * The first base sampler uses Peikert's inversion method, discussed in
      46             :  * section 4.1 of https://eprint.iacr.org/2010/088.pdf and summarized in
      47             :  * section 3.2.2 of
      48             :  * https://link.springer.com/content/pdf/10.1007%2Fs00200-014-0218-3.pdf.
      49             :  * Peikert's method requires precomputation of CDF tables around a specific
      50             :  * center and the table must be kept during the sampling process. Hence,
      51             :  * Peikert's method works best if the DESIRED STANDARD DEVIATION IS SMALL and
      52             :  * THE MEAN OF THE DISTRIBUTION IS FIXED, as each new center will require a new
      53             :  * set of precomputations.
      54             :  *
      55             :  * Second base sampler is  the Knuth-Yao Sampler discussed in section 5 of
      56             :  * https://link.springer.com/content/pdf/10.1007%2Fs00200-014-0218-3.pdf .
      57             :  * Similar to Peikert's, Knuth-Yao precomputes the PDF's of the numbers based on
      58             :  * standard deviation and the center, which is used during the sampling process.
      59             :  * Therefore like Peikert's method,  Knuth-Yao works best method works best if
      60             :  * the DESIRED STANDARD DEVIATION IS SMALL and THE MEAN OF THE DISTRIBUTION IS
      61             :  * FIXED, as each new center will require a new set of precomputations, just
      62             :  * like Peikert's inversion method.
      63             :  *
      64             :  * The "generic sampler" on the other hand, works independent from standard
      65             :  * deviation of the distribution. It combines an array of previously discussed
      66             :  * base samplers centered around 0 to (2^b-1) / 2^b through convolution. The
      67             :  * tables of base samplers however, must be precomputed beforehand; but they do
      68             :  * not need to be recalculated at any time of the sampling process. It is USABLE
      69             :  * FOR ANY STANDARD DEVIATION AND MEAN, just like Karney's method defined in
      70             :  * discretegaussiangenerator.h, needs only one single precomputation and is not
      71             :  * prone to timing attacks unlike Karney. Karney's method, however, is faster
      72             :  * than the generic sampler.
      73             :  *
      74             :  * PARAMETER SELECTION FOR GENERIC SAMPLER
      75             :  *
      76             :  * The selection of parameters change the run time/memory usage/precision of the
      77             :  * generic sampler. The triple trade off between these parameters are defined in
      78             :  * the equation k = (PRECISION - FLIPS) / LOG_BASE. k denotes the level of
      79             :  * precision of the generic sampler. Higher the k is, higher the precision of
      80             :  * the generic sampler but higher the run time. PRECISION denotes the number of
      81             :  * decimal bits in the center of the distribution. Since we are using 'double'
      82             :  * for mean, it is fixed to 53 by definition. FLIPS denote the number of
      83             :  * Bernoulli flips used to approximate the bits used in combination of base
      84             :  * sampler. Higher the number of flips, larger the number of bits approximated
      85             :  * rather than calculated which means smaller run times. Generic sampler
      86             :  * requires a set of base samplers centered around 0/2^b to (2^b-1)/2^b;
      87             :  * LOG_BASE denotes b in this equation. Higher the LOG_BASE is, more base
      88             :  * samplers required which requires additional memory; but at the same time
      89             :  * smaller run times.
      90             :  *
      91             :  * The base samplers used in generic sampler requires varying centers between
      92             :  * 0/2^b and (2^b-1)/(2^b) with the same standard deviation. The standard
      93             :  * deviation required for base samplers must satisfy SIGMA>=4*SQRT(2)*N, where
      94             :  * sigma is the standard deviation of the base sampler and N is the smoothing
      95             :  * parameter
      96             :  *
      97             :  * */
